# 🎨 Glyphor Designs - Graphic Designer Portfolio Website

A modern, sleek, and minimalistic portfolio website built with **React 19**, **Tailwind CSS v4**, and **React Router**, designed to showcase a graphic designer's creative work. Deployed with **Cloudflare Pages**, it delivers smooth performance, clean animations, and an elegant UI that enhances visual storytelling.

## 🚀 Project Overview

The **Graphic Designer Portfolio Website** is a fully responsive, dark-themed portfolio built to present creative works like **logos**, **illustrations**, **social media graphics**, and **posters**. It leverages the latest React and Tailwind releases, coupled with Cloudflare's global infrastructure for blazing-fast delivery.

Aimed at elevating the designer's professional presence online, the project focuses on clarity, visual impact, and seamless interaction. Subtle animations and intuitive navigation ensure visitors can engage with the work effortlessly.

---

## 🧩 Features

* 🎨 Clean, minimal layout focused on visuals
* ⚡ Blazing fast with **React 19** and **Vite**
* 🔁 Dynamic routing with **React Router**
* 🖼️ Image showcase grids
* 📱 Fully responsive for mobile, tablet, and desktop
* ✨ Framer Motion-style smooth entry animations
* 🌐 Deployed on **Cloudflare Pages**
